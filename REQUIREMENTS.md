# 要求定義書: インタラクティブ母音フォルマント合成器

## 1. プロジェクト概要

### 1.1 目的
御水（みもい）氏 (@mimoi_sound) が p5.js で作成した**インタラクティブ母音フォルマント合成器**を再現する。2D母音空間チャート上でマウスをドラッグすると、その位置に対応する母音がリアルタイムで合成・発音されるWebアプリケーション。

### 1.2 参考元
- **ライブデモ**: https://editor.p5js.org/mimoi/full/xBwV_oqCE
- **ソースコード**: https://editor.p5js.org/mimoi/sketches/xBwV_oqCE
- **元ネタ**: Golan Levin 氏の "vocal-vowel-formant-tonejs" (https://editor.p5js.org/golan/sketches/El9vQ13I1)
- **母音配置**: Dr. Geoff Lindsey の提唱する母音空間に基づく
- **歴史的オマージュ**: Gunnar Fant の OVE 合成音声デモ（"How are you?", "I love you"）

### 1.3 理論的背景
Gunnar Fant が提唱した**音源フィルタ理論** (Source-Filter Theory) に基づく。人間の声の生成過程を「音源（声帯振動）」と「フィルタ（声道の共鳴）」に分離してモデル化する。母音の違いは主に第1フォルマント (F1) と第2フォルマント (F2) の周波数の組み合わせで決まる。

- **F1**: 舌の高さ・口の開き具合に対応（低い＝閉じた口、高い＝開いた口）
- **F2**: 舌の前後位置・唇の丸めに対応（高い＝前舌、低い＝後舌）

## 2. 機能要件

### 2.1 音声合成エンジン

#### 2.1.1 音源（グロッタルパルス近似）
- **鋸歯状波オシレータ** (Sawtooth) を音源として使用
- 基本周波数: **131 Hz**（C3相当）
- マウス押下時に発音開始、リリース時に発音停止
- 振幅のフェードイン/フェードアウト: **0.1秒**で滑らかに遷移
- 最大振幅: **0.002**（小さい値でクリッピングを防止）

#### 2.1.2 フィルタチェーン（声道モデル）
オシレータの出力をメイン出力から切断し、以下の**6つのフィルタに並列接続**する:

| フィルタ名 | 種類 | 周波数 | Q値 | ゲイン | 制御 |
|-----------|------|--------|-----|--------|------|
| F1F | ピーキング (peaking) | 250–1000 Hz | 0.2 | 60 | マウスY座標で動的制御 |
| F2F | ピーキング (peaking) | 540–2600 Hz | 0.4 | 60 | マウスX座標で動的制御 |
| F3F | ピーキング (peaking) | 2500 Hz (固定) | 0.8 | 50 | 固定 |
| F4F | ピーキング (peaking) | 3500 Hz (固定) | 1.0 | 40 | 固定 |
| BEF | ノッチ (notch) | 5000 Hz (固定) | 0.2 | — | 固定（反フォルマント） |
| LPF | ローパス (lowpass) | 7250 Hz (固定) | 0.001 | — | 固定 |

### 2.2 母音空間チャート（ビジュアル）

#### 2.2.1 キャンバスとレイアウト
- キャンバスサイズ: **500 x 400 px**
- マージン: 上下左右 **50px**
- 背景色: 薄いグレー
- プロットエリア: 白色の矩形

#### 2.2.2 軸の仕様
- **X軸**: F2（第2フォルマント）、範囲 540–2600 Hz
  - **右から左へ**増加（音声学の慣例に従う: 左側が高F2 = 前舌母音）
- **Y軸**: F1（第1フォルマント）、範囲 250–1000 Hz
  - **上から下へ**増加（音声学の慣例に従う: 上が低F1 = 閉母音）
- 両軸とも**対数スケール** (`Math.log` / `Math.exp` による変換)
- 100 Hz 間隔でグリッド線を描画

#### 2.2.3 母音シンボルの配置
以下の **21個のIPA母音記号** をそれぞれの正準 F1/F2 位置にプロットする:

**基本母音 (Cardinal Vowels)**:
- i, e, ɛ, a（前舌系列）
- ɑ, ɔ, o, u（後舌系列）

**二次母音・中舌母音**:
- y, ø, œ（前舌円唇）
- ʌ, ɤ, ɯ（後舌非円唇）

**中央母音・その他**:
- ə（シュワー）
- ɵ, ɐ（中央系）
- ɪ, ʊ（近閉母音）
- æ, ɒ（開母音系）

#### 2.2.4 母音間の接続線
- 関連する母音同士を線で結び、伝統的な**母音四角形（vowel trapezoid）** / 母音コンステレーションを形成
- 前舌母音系列、後舌母音系列をそれぞれ縦に接続
- 円唇/非円唇ペアを水平に接続

#### 2.2.5 不可能領域
- F1 > F2 となる領域（右下）をグレーの三角形で塗りつぶし、物理的に不可能な領域として表示

#### 2.2.6 カーソル追従
- マウス位置に小さい円を描画し、現在の合成位置を視覚的に示す

### 2.3 ユーザーインタラクション

#### 2.3.1 基本操作
- **マウス押下 (mousePressed)**: 発音開始。オシレータの振幅を0.1秒かけて0.002までランプアップ
- **マウスドラッグ**: F1F と F2F のフィルタ周波数をリアルタイム更新。母音が連続的にモーフィング
- **マウスリリース (mouseReleased)**: 発音停止。振幅を0.1秒かけて0にランプダウン
- タッチデバイスでも同様の操作が可能であること

#### 2.3.2 座標-周波数マッピング
マウスのピクセル座標と周波数の変換には対数マッピングを使用:
- `pixelToHz(pixel, minHz, maxHz, minPixel, maxPixel)`: ピクセル→Hz変換
- `hzToPixel(hz, minHz, maxHz, minPixel, maxPixel)`: Hz→ピクセル変換

## 3. 非機能要件

### 3.1 技術スタック
- **フレームワーク**: p5.js (v1.9.0 以上)
- **オーディオ**: p5.sound アドオン（Web Audio API ベース）
- **言語**: JavaScript
- **ホスティング**: 静的ファイルとして配信可能（HTML + JS + CSS）

### 3.2 パフォーマンス
- フォルマントフィルタの更新がマウス移動に対してリアルタイムに追従すること
- 音声のレイテンシが知覚できないレベルであること（< 20ms目標）
- フェードイン/フェードアウトによりクリックノイズが発生しないこと

### 3.3 互換性
- PC環境推奨（マウス操作が前提）
- モダンブラウザ（Chrome, Firefox, Safari, Edge）で動作
- タッチデバイスでも基本操作が可能

### 3.4 ファイル構成
```
index.html    — p5.js, p5.sound の読み込みとスケッチの起動
sketch.js     — 全合成ロジックと描画ロジック（約200行）
style.css     — フルブリードキャンバスのスタイリング
```
