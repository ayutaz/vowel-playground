# 音源フィルタ理論の技術詳細

## 1. 音源フィルタ理論 (Source-Filter Theory)

Gunnar Fant (1960) が体系化。音声生成を2要素に分離:
- **音源 (Source)**: 声帯振動 → グロッタルパルス列（基本周波数F0の整数倍倍音を含む）
- **フィルタ (Filter)**: 声道共鳴 → 特定周波数帯を増幅（= フォルマント）

本実装: 鋸歯状波オシレータ (音源) + ピンクノイズ (気息性) → 6並列フィルタ (F1F, F2F, F3F, F4F, BEF, LPF)

## 2. グロッタルパルスの近似手法比較

| 手法 | スペクトル傾斜 | 自然さ | 計算コスト | 本実装 |
|------|-------------|-------|-----------|--------|
| インパルス列 | 0 dB/oct | 低（金属的） | 極低 | 不採用 |
| **鋸歯状波** | **-6 dB/oct** | **中** | **低** | **採用** |
| LFモデル | -12 dB/oct | 高 | 中~高 | 将来候補 |

**鋸歯状波が適切な理由**:
1. Web Audio API ネイティブ対応 (`type: 'sawtooth'`)
2. 全整数次倍音を含み、フォルマントフィルタの素材として十分
3. LPF (7250Hz) との組み合わせで実効 -18 dB/oct に近い高周波減衰

## 3. フィルタ構成と音質改善

### 6フィルタの役割

| フィルタ | 役割 | Q値設計意図 |
|---------|------|------------|
| F1F (peaking, 動的) | 舌の高さ = 母音の開閉 | Q=0.2（広帯域 = 咽頭の広い共鳴） |
| F2F (peaking, 動的) | 舌の前後 = 母音の前後 | Q=0.4（やや狭帯域 = 口腔前部の共鳴） |
| F3F (peaking, 動的) | 声の「存在感」、円唇性 | Q=0.8（高次モードのシャープな共鳴） |
| F4F (peaking, 動的) | 声質の「個人差」 | Q=1.0（さらにシャープ） |
| BEF (notch, 動的) | 反フォルマント = 高周波吸収 | Q=0.2（広く浅いスペクトル整形） |
| LPF (lowpass, 動的) | 声道の高周波減衰模倣 | Q=0.001（滑らかなロールオフ） |

**注**: F3F, F4F, BEF, LPF はベース周波数が固定（2500, 3500, 5000, 7250 Hz）だが、フォルマントスケーリング係数が乗算されるため、プリセットに応じて動的に変化する。

### Q値の段階的増加の根拠
音声学の実測データに基づく。低次フォルマント（F1, F2）は広い帯域幅、高次フォルマント（F3, F4）は狭い帯域幅を持つ。

## 4. 基本周波数 131 Hz (C3) の選択理由

1. **成人男性の話声位中心** (85-180Hz の中央付近)
2. **音楽的に切りの良い音高** (C3)
3. **十分な倍音密度**: 131Hz間隔の倍音列がF1/F2範囲に多数存在し、フォルマントフィルタが有効に機能
4. **フォルマント解像度**: F0が低いほど倍音間隔が狭く、フォルマントピークの近似が精密に

現在はプリセットごとにF0を変更可能（131～330Hz）。詳細は §8 を参照。

## 5. ホーミー（喉歌）的に聞こえる理由

CSVで複数ユーザーが指摘:
> 「ホーミーみたいな倍音聞こえんのおもろい」「倍音めっちゃ聞こえる!!」

**技術的原因**:
1. **完全周期性**: デジタルオシレータはジッター（F0揺らぎ）がゼロ。自然な声は ~0.5-1% のジッターで倍音が曖昧になるが、合成音では完全にシャープ
2. **高Q値フィルタ**: F3F (Q=0.8), F4F (Q=1.0) が特定倍音を目立たせる → ホーミーと同じ原理
3. **固定F0**: 自然発話のF0微小変動がなく、倍音可聴性が極めて高い
4. **フォルマント連続移動**: ドラッグで増幅される倍音が次々切り替わり「メロディ的な倍音移動」が知覚される

**改善策（実装済み）**:
- ビブラート（5Hz, ±3-5Hz）でF0の微小変動を付与（§10 参照）
- ピンクノイズ混合でブレシネス（気息性）を付加（§9 参照）

ただし、通常プリセットでは従来通りのホーミー的音質を維持している（ユーザーの関心を引き、ツイート拡散に貢献している面があるため）。

## 6. 二重母音の自然な合成

マウスドラッグによるフォルマント連続遷移で、二重母音が自然に生じる。

| 遷移速度 | 所要時間 | 知覚される音 |
|---------|---------|------------|
| 極めて速い | < 30 ms | 子音的 (/w/, /j/) |
| 速い | 50-100 ms | 自然な二重母音 |
| 中程度 | 100-200 ms | ゆっくりした二重母音 |
| 遅い | > 300 ms | 2つの別個の母音 |

ユーザー報告: 「u→aを高速で動かしたら "wa" みたいになる」— /u/→/a/ の高速遷移は実際の /wa/ のフォルマント遷移パターンと一致。半母音 /w/ が本質的に /u/ からの急速遷移であるという音声学の知見を体感的に実証。

## 7. フォルマントスケーリングと声道長正規化 (VTLN)

### 理論的背景

声道長正規化 (Vocal Tract Length Normalization, VTLN) は、声道の物理的な長さの違いによるフォルマント周波数のシフトを統一的に扱う手法である。声道が短いほど共鳴周波数が高くなるという音響管の物理法則に基づく。

**声道長とフォルマント周波数の関係**:
フォルマント周波数は声道長に反比例する。すなわち:

```
F_n ∝ (2n - 1) * c / (4 * L)
```

ここで `c` は音速（約340 m/s）、`L` は声道長、`n` はフォルマント次数。

### 声道長カテゴリとスケーリング係数

文献値に基づく声道長とスケーリング比:

| カテゴリ | 声道長 | 基準男性比 | スケーリング係数 |
|---------|--------|-----------|---------------|
| 成人男性 | ~17.6 cm | 1.00 | 1.00 |
| 成人女性 | ~15.6 cm | 0.89 | **×1.13** |
| 児童 | ~12.0 cm | 0.68 | **×1.35–1.50** |

### 本実装でのスケーリング適用

全フォルマント周波数にスケーリング係数 `scale` を一律乗算:

```javascript
F1F.set(yToF1(py) * scale, q[0]);    // 動的フォルマント
F2F.set(xToF2(px) * scale, q[1]);    // 動的フォルマント
F3F.set(2500 * scale, q[2]);         // 固定フォルマント
F4F.set(3500 * scale, q[3]);         // 固定フォルマント
BEF.set(5000 * scale, 0.2);          // 反フォルマント
LPF.set(7250 * scale, 0.001);        // 高周波減衰
```

プリセット別のスケーリング係数:

| プリセット | F0 (Hz) | scale | 対応する声道長概算 |
|-----------|---------|-------|-----------------|
| 通常 | 131 | 1.00 | ~17.6 cm（成人男性） |
| 女声 | 220 | 1.15 | ~15.3 cm（成人女性相当） |
| 可愛い声 | 280 | 1.30 | ~13.5 cm（小柄な女性/少女相当） |
| アニメ声 | 330 | 1.50 | ~11.7 cm（幼児相当） |

F0の上昇だけでは「高い声」にしかならないが、フォルマントスケーリングを組み合わせることで「小さい身体から出る声」の物理的な特性を再現できる。これがVTLNの本質的な効果である。

## 8. 可愛い声の音響的特性 — "Super Kawaii Vocalics" に基づく知見

### 参照研究

"Super Kawaii Vocalics" (arXiv:2507.06235, CHI 2025) は、日本のアニメ・ゲームにおける「可愛い声」の音響パラメータを統計的に分析した研究である。

### F0と可愛さの相関

**F0（基本周波数）は可愛さ知覚と最も強い相関を示す**: rs = .89

これは直感的にも理解しやすい。高い声 = 小さい身体 = 幼い/可愛いという連想が働く。

### F1と可愛さの相関

**F1もまた有意な相関を示す**: rs = .74

F1は口の開きに対応するため、可愛い声では口を小さく開ける傾向（F1が低め）があることを示唆する。これは「もごもご」した、やや閉じた口での発話パターンと一致する。

### F0による声質カテゴリ

研究および実践から導出された声質分類:

| カテゴリ | F0範囲 | 声質の特徴 | 代表的なキャラクター類型 |
|---------|--------|-----------|---------------------|
| **ロリ声** | ≥350 Hz | 極めて高く、幼い印象 | 幼女、マスコット |
| **萌え声** | 300–350 Hz | 甘く、明るい | 妹系、アイドル |
| **少女声** | 280–300 Hz | 若々しく、活発 | 女子高生、ヒロイン |
| **お姉さん声** | 250–280 Hz | 落ち着いた女性らしさ | お姉さん、先輩 |
| **大人声** | <250 Hz | 成熟、知的 | 大人の女性、母親 |

本実装のプリセットとの対応:
- **通常** (F0=131Hz): 成人男性話声（カテゴリ外）
- **女声** (F0=220Hz): 大人声カテゴリ付近
- **可愛い声** (F0=280Hz): 少女声カテゴリ
- **アニメ声** (F0=330Hz): 萌え声カテゴリ

## 9. ブレシネス（気息性）制御 — ピンクノイズ混合

### HNR (Harmonics-to-Noise Ratio) の概念

自然な声は、周期的な声帯振動（調波成分）と非周期的な乱流ノイズ（気息成分）の混合である。HNR（調波対雑音比）はこの比率を表す:
- **HNR が高い** → クリアな声（調波成分が支配的）
- **HNR が低い** → 息混じりの声（ノイズ成分が多い）

可愛い声はしばしば軽い気息性を伴う。これは声門を完全に閉じないウィスパー気味の発声パターンに起因する。

### 実装

ピンクノイズ（`p5.Noise("pink")`）を音源に混合:

```javascript
noise = new p5.Noise("pink");
noise.amp(AMP * preset.breath * 0.5, FADE_TIME);
```

**ピンクノイズを選択した理由**: ピンクノイズは -3 dB/oct のスペクトル傾斜を持ち、自然な気息ノイズのスペクトル特性に近い。ホワイトノイズ（0 dB/oct）では高周波が強すぎて不自然になる。

プリセット別の breath パラメータ:

| プリセット | breath | 気息性の程度 |
|-----------|--------|------------|
| 通常 | 0 | なし（完全周期的） |
| 女声 | 0.15 | わずかな気息性 |
| 可愛い声 | 0.20 | 軽いウィスパー感 |
| アニメ声 | 0.25 | 明瞭な気息性 |

ノイズも鋸歯状波と同じ6並列フィルタチェーンに接続されるため、フォルマント構造を持つ気息成分となる（単純なホワイトノイズ付加ではなく、声道フィルタを通過した気息ノイズの模倣）。

## 10. ビブラートによる自然性の付与

### ビブラートの音響的意味

自然な声にはF0の微小な周期的変動（ビブラート）と非周期的変動（ジッター）が存在する。完全に一定のF0はロボット的/電子的に聞こえる（§5のホーミー問題の原因の一つ）。

### 実装パラメータ

```javascript
let vibOffset = preset.vibDepth * Math.sin(TWO_PI * preset.vibRate * millis() / 1000);
osc.freq(preset.f0 + vibOffset);
```

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| ビブラート周波数 (vibRate) | 5.0–5.5 Hz | 歌唱音声のビブラート典型値（4-7 Hz） |
| ビブラート深さ (vibDepth) | 0–4 Hz (±) | 話声は ±3-5 Hz が自然。歌唱では ±数十Hzにもなる |

プリセット別ビブラート設定:

| プリセット | vibRate | vibDepth | 効果 |
|-----------|---------|----------|------|
| 通常 | 5.5 Hz | 0 Hz | なし（従来互換） |
| 女声 | 5.5 Hz | ±1.5 Hz | 微かな揺らぎ |
| 可愛い声 | 5.0 Hz | ±3 Hz | 自然な話声の揺らぎ |
| アニメ声 | 5.0 Hz | ±4 Hz | やや強調された揺らぎ |

ビブラートは正弦波変調で実装しているが、自然な声のジッターは不規則である。将来的にはランダム成分の追加（シマー/ジッター）でさらに自然性を向上できる。

## 11. スペクトルティルト制御 — 可愛い声の明るさ

### スペクトルティルトとは

スペクトルティルト（spectral tilt）は、倍音列の高周波側がどの程度減衰するかを示す指標。典型的な声のスペクトルは高域ほどエネルギーが減衰する（負の傾斜）。

- **急な傾斜（steep tilt）**: 高周波が弱い → 暗く、こもった声
- **平坦な傾斜（flat tilt）**: 高周波が相対的に強い → 明るく、鮮明な声

### 可愛い声とスペクトルティルトの関係

可愛い声は一般にスペクトルティルトが平坦（明るい声質）である。本実装ではフォルマントフィルタのゲイン配分でこれを制御する。

**通常プリセット（急な傾斜 = 暗い声）**:
```
F1: 60 dB → F2: 60 dB → F3: 50 dB → F4: 40 dB   (高域ほど大きく減衰)
```

**アニメ声プリセット（平坦な傾斜 = 明るい声）**:
```
F1: 48 dB → F2: 52 dB → F3: 58 dB → F4: 55 dB   (F3/F4が相対的に強い)
```

ゲイン推移の比較:

| プリセット | F1 gain | F2 gain | F3 gain | F4 gain | 傾斜の特徴 |
|-----------|---------|---------|---------|---------|-----------|
| 通常 | 60 | 60 | 50 | 40 | 急傾斜（-20 dB差） |
| 女声 | 58 | 58 | 55 | 45 | やや緩和（-13 dB差） |
| 可愛い声 | 52 | 55 | 58 | 52 | ほぼ平坦（F3がピーク） |
| アニメ声 | 48 | 52 | 58 | 55 | 逆傾斜（F3/F4 > F1/F2） |

可愛い声・アニメ声プリセットではF3/F4のゲインがF1/F2を上回る「逆傾斜」に設定されている。これにより、スペクトルの高域が相対的に強調され、明るく鮮明な声質が実現される。

## 12. プリセット設計の総合的な考え方

各プリセットは以下の5つのパラメータ軸を組み合わせて声質を制御する:

| パラメータ軸 | 物理的な対応 | 可愛さ方向の変化 |
|-------------|------------|----------------|
| F0（基本周波数） | 声帯振動速度 | ↑ 高く（小さい声帯） |
| scale（フォルマントスケーリング） | 声道の短さ | ↑ 大きく（短い声道） |
| breath（気息性） | 声門閉鎖の不完全さ | ↑ 多く（ウィスパー感） |
| vibDepth（ビブラート深さ） | F0の揺らぎ | ↑ 大きく（自然な揺れ） |
| gain 配分（スペクトルティルト） | 声道の共鳴特性 | F3/F4を相対的に強調（明るさ） |

これらは独立ではなく、相互に関連する。例えばF0を上げると倍音間隔が広がるため、フォルマントフィルタの効きが変わる。スケーリング係数はF0の変化と連動させることで、身体サイズの変化として一貫した知覚を生む。

**設計原則**: 単一パラメータの極端な変更ではなく、複数パラメータの協調的変化により、物理的に妥当な声質変化を実現する。
